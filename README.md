> The web application front-end for IoTprint project

### ✨ [Demo](https://iotprint.netlify.app)

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

## Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
